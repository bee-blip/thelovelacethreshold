The Lovelace Threshold

A diagnostic to examine a rule, policy, or accountability instrument and finds where it has gone blind to what actually produces the outcomes it governs. It's point at which it stays formally valid but is no longer enough.

Assessment Use Only: Answer for the instrument you are checking. This is generic by design; the same logic applies to any governed system.
